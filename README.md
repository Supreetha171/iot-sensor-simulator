# IoT Sensor Data Simulator

This project simulates an IoT sensor device using Python.

The program generates random temperature and humidity data and sends it to the ThingSpeak cloud platform using REST APIs.

The data is visualized in real-time using ThingSpeak dashboards.

## Technologies Used
- Python
- ThingSpeak API
- REST API
- Cloud Data Visualization

## Features
- Virtual sensor data generation
- Real-time cloud data upload
- Live graph visualization

## Example Output
Temperature: 33
Humidity: 74
